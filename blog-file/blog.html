Question: Discuss the scope of var, let, and const

Answer: The var, let, and const is the keyword to declare variables in javascript. The var variable is an old method to declare a variable in javascript. In modern javascript, we use the let and const variable, which was introduced in the ES2015(ES6) update; now, the let and const variable is used more frequently in modern javascript as compared to the var variable.

Question: Tell us the use cases of null and undefined ?

Answer: A null means the absence of a value. A variable is undefined when you haven't assigned any value yet, not even a null.
